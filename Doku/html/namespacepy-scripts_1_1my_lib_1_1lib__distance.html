Documentation for method simple_measure(int:pin ,int:pin) Vollführt eine einfache Messung über einen Schallabstandssensorr auf angegebenen Pins.

Dabei wird eine Signal vom Trigger ausgestoßen, einen kurzen Moment gewartet unf dann der Zeitabstand bis zum reflektierten Eingang des Signals abgewartet. Aus diesem Wert wird dann der Abstand in cm berechnet. Formel: Abstand = (Zeitunterschied * Schallgeschwindigkeit)/2
