First things First This is a control-software, build to control different Drones, Rovers and things alike. The blue-prints for these devices aren't online now and probably never will.

The actual init can be found in main/py-scripts The webinterface can be found in main, called Index.php

Preparation For using the webinterface you need a php-runner.
